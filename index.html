// 진단 방식

3분 안에, 당신의 판단 패턴이 드러납니다

24개의 문항을 통해 당신이 빨리 정하는 사람인지, 직감으로 읽는 사람인지, 기준을 세우는 사람인지, 바로 실행하는 사람인지를 읽어냅니다.

STEP 01 // 응답

24개 행동 성향 문항

압박 상황, 불확실한 정보, 갈등 장면에서 당신이 실제로 어떻게 판단하는지를 묻습니다. 5점 척도, 한 문항씩 집중 응답.

STEP 02 // 분석

4개의 판단 축 분석

빠르게 결정하는지, 직감으로 보는지, 구조를 세우는지, 바로 실행하는지를 4개의 판단 축으로 읽어냅니다. 그 조합이 당신의 판단 캐릭터가 됩니다.

STEP 03 // 결과

나의 판단 캐릭터 공개

캐릭터명 · 정의 · 강점 · 주의점이 공개됩니다. 공유 카드로 친구와 비교하고, 유료 TP-IQ로 더 깊이 열어볼 수 있습니다.

// Why TP-IQ

왜 군 전술가 기준으로 보나요?

판단지능은 단순한 성향이 아닙니다. 압박, 불확실성, 제한된 정보 속에서 어떻게 읽고, 어떻게 결단하고, 어떻게 실행으로 옮기는지를 봅니다. TP-IQ는 이런 판단 구조를 군 전술가 관점의 9개 차원으로 해석합니다.

// MBTI와의 차이
MBTI는 '나는 어떤 성향인가'를 봅니다.
TP-IQ는 '나는 압박 속에서 어떻게 판단하는가'를 측정합니다.
성격이 아닌 판단 구조의 차이입니다.

MBTI가 Jung의 심리유형 이론을 바탕으로 사람의 선호 차이를 설명하듯, 판단지능은 한 가지 성격이론이 아니라, 의사결정 심리학과 자연주의적 판단 연구를 결합한 프레임 위에서 설계되었습니다.

TP-IQ 무료 판단지능은 네 가지 축으로 당신의 판단 패턴을 읽습니다:

①빠르게 결정하는가, 충분히 숙고하는가
②직감으로 읽는가, 근거를 모으는가
③구조를 세우는가, 상황에 맞게 바꾸는가
④정면 돌파하는가, 조율하며 운영하는가

이 네 축은 다음 연구 전통에서 이론적 근거를 얻습니다:

Dual-Process Theory
빠른 직관적 처리(System 1)와 느린 분석적 처리(System 2)의 구분 — Kahneman(2011)
Recognition-Primed Decision (RPD)
시간 압박 속 경험 기반 신속 판단 모델 — Klein(1998)
Situation Awareness (SA)
상황 인식·이해·예측의 3단계 능력 — Endsley(1995)
Cognitive Flexibility
변화 조건에 맞춰 사고·행동을 조정하는 능력 — Boyd OODA Loop(1987)
